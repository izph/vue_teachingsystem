# 教学管理系统(该项目功能简单，还有待完善)

该项目以老师为中心设计，老师管理着不同班级的学生，可以修改学生的基本信息（增删改查），同时老师也可以修改自己个人信息，头像，登录密码，老师还可以对学生进行点名，课堂抽查等。
# 核心技术栈

vue.js+Node.js+mongodb

## 服务端项目搭建
```
npm install
```
## 服务端运行
```
node index.js
```
## 客户端项目搭建
```
npm install
```
### 客户端运行
```
npm run serve
```
